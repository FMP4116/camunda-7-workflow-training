# Instalar extensiones de VS Code

## 🎯 Objetivo

Preparar el entorno de desarrollo instalando las extensiones necesarias para trabajar con:

* Java
* Maven
* Spring Boot
* BPMN
* Camunda

Estas extensiones facilitarán la edición de código, la ejecución del proyecto y el modelado de procesos.

---

## 🧠 Contexto

Durante el curso trabajaremos con:

* **Java 17**
* **Spring Boot**
* **Maven**
* **Camunda 7**
* **BPMN**

Visual Studio Code puede ampliarse mediante **extensiones** que añaden soporte específico para estas tecnologías.

Instalar estas extensiones desde el principio evita muchos problemas durante los laboratorios.

---

## 📍 Abrir el panel de extensiones

En VS Code abrir el panel de extensiones:

```
Ctrl + Shift + X
```

O seleccionar el icono:

```
Extensions
```

en la barra lateral izquierda.

---

## 📦 Extensiones a instalar

Instalar las siguientes extensiones.

### Extension Pack for Java

Buscar:

```
Extension Pack for Java
```

Publicador:

```
Microsoft
```

Instalar.

Este paquete incluye:

* soporte para Java
* depuración
* autocompletado
* gestión de proyectos Maven

---

### Spring Boot Extension Pack

Buscar:

```
Spring Boot Extension Pack
```

Publicador:

```
VMware
```

Instalar.

Permite:

* ejecutar aplicaciones Spring Boot
* navegación por beans
* soporte para configuración

---

### BPMN Editor

Buscar:

```
BPMN Editor
```

Publicador:

```
bpmn-io
```

Instalar.

Esta extensión permite:

* visualizar archivos `.bpmn`
* editar procesos BPMN desde VS Code

---

### Maven for Java

Buscar:

```
Maven for Java
```

Publicador:

```
Microsoft
```

Instalar.

Permite:

* ejecutar comandos Maven
* gestionar dependencias
* compilar el proyecto

---

## ✅ Verificación

Una vez instaladas las extensiones:

1. Abrir el panel de **Extensiones instaladas**
2. Verificar que aparecen:

```
Extension Pack for Java
Spring Boot Extension Pack
Maven for Java
BPMN Editor
```

Si aparecen en la lista, el entorno está listo para continuar con el siguiente paso.
